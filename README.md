# Class Charts API

This is a services based API that provides a way to interact with the Class Charts platform.



## About

### Supported Authentication Types
 - **Student**
 - **[NOT IMPLEMENTED] Parent**
 - **[NOT IMPLEMENTED] Teacher**

### Supported Features
 - **Detentions**
 - **Timetable**
 - **Homework**

### Kotlin Multiplatform

Library is written in Kotlin Multiplatform. It can be used in many platforms like JVM, Android, iOS, JavaScript, Linux, Windows, macOS, and more.
